Excel avanzado

Aprende funciones, tablas dinámicas y Power Query

Curso: Excel avanzado en Udemy

Práctica: Crear una tabla dinámica que sume ventas por mes.

SQL

Practica con joins, subqueries y ventanas.

Curso: SQLZoo

Ejemplo de código:

SELECT customers.name, orders.total
      FROM customers
      JOIN orders ON customers.id = orders.customer_id
      WHERE orders.total > 100;

Python para análisis

Librerías clave: pandas, numpy, seaborn

Libro: "Python for Data Analysis" de Wes McKinney

Ejemplo: Leer un archivo CSV

import pandas as pd

# Leer archivo CSV
datos = pd.read_csv('ventas.csv')
print(datos.head())

Visualización de datos

Aprende con Power BI y Python (matplotlib, seaborn)

Curso recomendado: Coursera o edX

Ejemplo básico en Python:

import matplotlib.pyplot as plt

ventas = [250, 300, 400, 350]
meses = ['Enero', 'Febrero', 'Marzo', 'Abril']

plt.bar(meses, ventas)
plt.title('Ventas por Mes')
plt.show()
